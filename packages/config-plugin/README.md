# @chinafast/expo-wechat-auth-config-plugin

Expo config plugin used by `@chinafast/expo-wechat-auth`.

Most apps should install and configure `@chinafast/expo-wechat-auth` directly, then reference the plugin as `@chinafast/expo-wechat-auth/plugin` in `app.json`.

The plugin configures iOS URL schemes and Associated Domains, Android package visibility and deep links, and generates the `WXEntryActivity` required by the bundled native WeChat bridge.
